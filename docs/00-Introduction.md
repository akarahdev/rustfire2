# Installation
To install RustFire, you first must have `cargo` and `rustc` installed.
Start a new Rust project by using:
```sh
cargo init
```
Then, go into that directory and add RustFire as a dependency with:
```sh
cargo add rustfire
```
or, edit the `Cargo.toml` with:
```toml
[dependencies]
rustfire = "0.4.0"
```
